About this project

This is a project initiated by NISH, Trivandrum for the betterment of disabled people. Mainly this device serves as the Alet system in Library to Alert the dough and blind people using vibration.

"As a LIBRARY ALERT"

Features

Wireless Control device to control remotely.

Rechargeable battery to power continously.

Compact and ergonomic design to handle easily.

Week 1

Activities which we had gone through on week 1 includes:

1.We initially listed the components required to design the circuit.

2.Block diagram was drawn with the knowledge we got about the product. We had drawn 2 block diagrams which includes:

a) transistor switch with separated GND.

b) transistor switch with separated 5V.

3.Motor control logic development and testing

a) ESP32 LED blinking

b) ESP32 Motor control

c) ESP32 blynk app interface

d) ESP32 blynk app motor control
